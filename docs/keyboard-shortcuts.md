# ⌨️ Keyboard Shortcuts

This document lists the essential keyboard shortcuts for the `container-debian-desktop` environment.

## 🪟 Window Tiling (New!)
Use these shortcuts to quickly snap windows to different parts of the screen.

| Action | Shortcut |
| :--- | :--- |
| **Tile Left** | `Ctrl` + `Shift` + `←` |
| **Tile Right** | `Ctrl` + `Shift` + `→` |
| **Tile Up** | `Ctrl` + `Shift` + `↑` |
| **Tile Down** | `Ctrl` + `Shift` + `↓` |

---

## 🚀 System & Applications
Commonly used shortcuts for system navigation and launching tools.

| Action | Shortcut |
| :--- | :--- |
| **Toggle Guake Terminal** | `F12` |
| **Open Application Finder** | `Alt` + `F2` or `Alt` + `F3` |
| **Open Terminal** | `Ctrl` + `Alt` + `T` |
| **Open File Manager** | `Ctrl` + `Alt` + `F` |
| **Lock Screen** | `Ctrl` + `Alt` + `L` |
| **Logout / Shutdown** | `Ctrl` + `Alt` + `Delete` |
| **Force Kill Window** | `Ctrl` + `Alt` + `Escape` |

---

## 🖥️ Workspace Management
Navigate between your virtual desktops.

| Action | Shortcut |
| :--- | :--- |
| **Switch Workspace Left** | `Ctrl` + `Alt` + `←` |
| **Switch Workspace Right** | `Ctrl` + `Alt` + `→` |
| **Switch Workspace Up** | `Ctrl` + `Alt` + `↑` |
| **Switch Workspace Down** | `Ctrl` + `Alt` + `↓` |

---

## 🖼️ Window Management
Standard window controls.

| Action | Shortcut |
| :--- | :--- |
| **Close Window** | `Alt` + `F4` |
| **Maximize Window** | `Alt` + `F10` |
| **Minimize Window** | `Alt` + `F9` |
| **Cycle Windows (Tab)** | `Alt` + `Tab` |
| **Show Desktop** | `Ctrl` + `Alt` + `D` |

---

> **Note:** The **Windows Key (Super)** is often intercepted by your host operating system and may not work as expected within the browser-based VNC session. Custom shortcuts are designed to use `Ctrl`, `Alt`, and `Shift` for better compatibility.
